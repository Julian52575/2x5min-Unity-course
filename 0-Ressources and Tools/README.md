## Ressources and Tools 

> [!CAUTION]
> Some unity problems can be tricky to describe and even trickier to find ressources on.
> Your best bet is unfortunatly to try AI and hope you described the situation properly. If not, cross reference it with other blog posts related to the matter.     
> _(A personal example of mine was enabling lightnings in the main shader)._

#### AI chatbot

I personaly use ChatGPT or Claude for the following work:
- Coding simple component
- Explaining Unity's built-ins features or components
- Debugging

#### Search Engines

Mainly google.com. Google your problem and see what comes up.
The AI popup is surprinsingly good for Unity.

#### Unity scripting reference 

The list of build-in classes and be found in the reference [here](https://docs.unity3d.com/6000.5/Documentation/ScriptReference/GameObject.html).   
Use it to know more about the classes you need to use inside your scripts.  

> [!TIP]
> Install the Unity reference extension for your IDE to access it more easily.

#### Unity forum

The official forum can be found [here](https://discussions.unity.com/). 
Some other users might have found themselves in the same situation as you.  
Do note that most discussion are dated howerever.
